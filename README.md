# cs3773-group3 G
Game Vault is an online video game store.

## Features
- Browse digital games
- Browse physical games
- Browse merchandise
- Search, sort, and filter products
- View product detail pages
- Add products to cart
- Save orders through a Node/Express backend
- Manage users, games, merchandise, and discount codes through backend routes

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- JSON files for data storage
- Git and GitHub

## How to Run
1. Clone the repository.
2. Open the `Game Vault` folder.
3. Install dependencies:

```bash
npm install
```
4. Start the backend using "node server.js"
5. Open homePage/index.html using Live Server in VS Code.
